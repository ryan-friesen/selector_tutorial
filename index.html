The reset function here was the most complex to do- it involves several steps:

  • Collecting the list objects with the querySelectorAll() method
  • Using a for loop to cycle through the length of the returned value from said method
    • Within the for loop, cycling through the specific values of the collected list objects
    • Using the .push() method to insert these into an array, sorted by these values
    • Returning the array to the function that clears the innerHTML of the list within the first container
